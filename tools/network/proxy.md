## 服务端面板 

### [3x-ui](https://github.com/MHSanaei/3x-ui)
目前最流行的 Xray 面板（原版 x-ui 的增强分支），支持 VLESS Reality 等新协议。
* **一键安装脚本:**
```bash
bash <(curl -Ls [https://raw.githubusercontent.com/mhsanaei/3x-ui/master/install.sh](https://raw.githubusercontent.com/mhsanaei/3x-ui/master/install.sh))
```

## 订阅转换

### [[sub-web-modify]]
基于 Vue 开发的 Subconverter 订阅转换**前端**增强版。通常需要配合后端使用，用于将节点链接转换为 Clash/Sing-box 等客户端可识别的订阅链接。

* **项目地址:** [youshandefeiyang/sub-web-modify](https://github.com/youshandefeiyang/sub-web-modify)
* **主要特性:**
    * 预置了大量远程配置（ACL4SSR 等）
    * 界面优化，支持更多自定义选项
* **部署方式:** 通常通过 Docker 部署或作为静态网页托管。
